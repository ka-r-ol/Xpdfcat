GUI tool to merge and slice PDF files

# Content

# Installation

## prerequesites
    * pytho0n3
    * tkinter python module
    * PyPDF2 python module

## Install
### using git
git clone https://github.com/ka-r-ol/Xpdfcat
### manually
download [Xpdfcat.pyw]

# Usage
$ python3 Xpdfcat.pyw
