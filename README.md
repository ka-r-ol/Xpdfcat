GUI tool to merge and slice PDF files

# Content

# Installation

## prerequesites

    Components required in the enviroment:
    * python3
    with following modules:
      * tkinter
      * PyPDF2

## Install

### using git

```
git clone https://github.com/ka-r-ol/Xpdfcat
```
### manually

Download ![download](Xpdfcat.pyw)

# Usage

$ python3 Xpdfcat.pyw
