# Installation

## prerequesites

    Components required in the enviroment:
    * python3
    * tkinter python module
    * PyPDF2 python module

## Install

### using git
'''
git clone https://github.com/ka-r-ol/Xpdfcat
'''
### manually
Download !download [Xpdfcat.pyw]

# Usage
'''
$ python3 Xpdfcat.pyw
